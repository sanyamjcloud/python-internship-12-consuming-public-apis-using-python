# API Data Fetch using Python

## Overview
This project demonstrates how to fetch data from a free public API using Python.  
The program sends a GET request, processes the JSON response, handles errors, and stores the fetched data locally.

## Technologies Used
- Python
- requests library
- VS Code

## Features
- Sends GET request to a public API
- Checks HTTP response status codes
- Parses JSON response into Python dictionary
- Extracts required data from nested JSON
- Handles API and network related errors
- Displays clean and readable output
- Stores fetched data in a local JSON file

## How to Run
```bash
pip install -r requirements.txt
python api_data_fetch.py
